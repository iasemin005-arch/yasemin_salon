# 🌸 Ясемин — Сайт салона красоты

## Быстрый старт

### 1. Установить зависимости
```bash
pip install -r requirements.txt
```

### 2. Запустить сервер
```bash
python manage.py runserver
```

### 3. Открыть в браузере
- **Сайт:** http://127.0.0.1:8000
- **Админка:** http://127.0.0.1:8000/admin

### Логин в админке
- Логин: `admin`
- Пароль: `admin123`

## Что есть на сайте
- Главная страница с Hero-секцией
- Прайс-лист услуг с вкладками по категориям
- Галерея работ (добавляется через админку)
- Отзывы клиентов (одобряются в админке)
- Онлайн запись с выбором услуги, мастера, даты/времени
- Оплата наличными или картой
- Контакты и карта

## Структура проекта
```
yasemin_salon/
├── manage.py
├── requirements.txt
├── db.sqlite3          ← база данных
├── salon/              ← приложение
│   ├── models.py       ← модели (Service, Master, Appointment, Review...)
│   ├── views.py        ← логика страниц и API
│   ├── admin.py        ← настройка админки
│   └── urls.py         ← маршруты
├── templates/salon/    ← HTML шаблоны
├── static/             ← CSS, JS
└── media/              ← загруженные фото
```
